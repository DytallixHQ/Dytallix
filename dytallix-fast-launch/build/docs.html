Introduction

Welcome to the Dytallix developer documentation. This guide will help you build quantum-resistant blockchain applications.

What is Dytallix?

Dytallix is a blockchain platform built from the ground up with post-quantum cryptography (PQC). We implement NIST-approved algorithms including Kyber, Dilithium, SPHINCS+, and Falcon to ensure your applications remain secure in the quantum era.

Quick Start

Get up and running with Dytallix in 5 minutes.

Step 1: Install the SDK

# Using npm
npm install @dytallix/sdk

# Using yarn
yarn add @dytallix/sdk

Step 2: Initialize the Client

import { DytallixClient } from '@dytallix/sdk';

const client = new DytallixClient({
  endpoint: 'https://testnet.dytallix.com',
  apiKey: 'your_api_key'
});

Step 3: Create a Wallet

const wallet = await client.createWallet({
  algorithm: 'dilithium',
  securityLevel: 3
});

console.log('Address:', wallet.address);

REST API Reference

Complete OpenAPI-compatible REST API documentation.

Authentication

All API requests require an API key in the header:

Authorization: Bearer YOUR_API_KEY

Endpoints

POST /api/v1/wallet/create

Create a new PQC wallet

GET /api/v1/explorer/block/:height

Get block information by height

GraphQL API

Type-safe queries with GraphQL. Endpoint: https://api.dytallix.com/graphql

Example Query

query GetLatestBlocks {
  blocks(limit: 10, orderBy: HEIGHT_DESC) {
    height
    hash
    timestamp
    transactions {
      hash
      signatureType
    }
  }
}
