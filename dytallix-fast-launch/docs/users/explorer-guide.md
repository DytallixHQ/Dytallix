# Explorer Guide

Use the explorer to inspect blocks and transactions.

> Last updated: 2025-08-20

## Search

Enter a block height, transaction hash, or address in the search bar.

## Verify Transaction

1. Submit a transaction.
2. Visit `https://explorer.dytallix.example/tx/<hash>`.
3. Check status and events.

Permalinks follow `/tx/<hash>` and `/address/<addr>` patterns.
