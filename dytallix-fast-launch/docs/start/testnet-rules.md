# Testnet Rules

Guidelines for using the Dytallix public testnet.

> Last updated: 2025-08-20

1. Tokens have **no monetary value**.
2. Network resets occur first Monday of each month; all state wiped.
3. Faucet limited to 100 tokens per day per address.
4. Requests and transactions are logged for abuse prevention.
5. Data retention is 30 days.
6. Use responsibly; no spam or illegal content.
7. Support at [Discord](../community/community-channels.md#discord).
