# Prelaunch Final Validation Evidence

This directory contains evidence artifacts generated by the prelaunch validation script.

## Contents

When `scripts/prelaunch_validation.sh` runs, it generates:

- **logs/** - Service bootstrap logs
- **json/** - Transaction receipts, wallet addresses, balance snapshots
- **governance/** - Proposal submission, votes, execution logs
- **wasm/** - Smart contract deployment and execution receipts
- **ai/** - AI risk oracle query results
- **ports.env** - Service port configuration
- **SUMMARY.md** - Comprehensive validation summary report

## Usage

Run the validation script:

```bash
# Full validation with live services
./scripts/prelaunch_validation.sh

# Mock mode (generate evidence without starting services)
./scripts/prelaunch_validation.sh --mock
```

The script will automatically generate all evidence artifacts in this directory.

## Notes

- Evidence files are **not tracked by git** (see `.gitignore`)
- Each run generates fresh evidence with current timestamps
- Wallet addresses are redacted in saved artifacts for security
- All JSON files are structured for automated parsing

For more details, see `scripts/README_PRELAUNCH_VALIDATION.md`.
