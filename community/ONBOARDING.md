# Dytallix Contributor Onboarding Guide

Welcome to Dytallix! This guide will help you get started as a contributor.

## Steps
1. Read the vision and context docs in `documents/`
2. Review the architecture in `ARCHITECTURE.md` and `API_REFERENCE.md`
3. Pick a module (blockchain, AI, wallet, contracts, etc.)
4. Clone the repo and set up your dev environment
5. See `CONTRIBUTING.md` for workflow and code style
6. Join discussions and submit proposals in `community/proposals/`
7. Ask questions and suggest improvements!
