# Community Channels

Stay connected during the public testnet.

| Channel | Purpose |
|---------|---------|
| Discord | Real-time chat, support |
| Forum (roadmap) | Long-form proposals, RFCs |
| GitHub Issues | Bug reports, feature requests |
| Status Page (roadmap) | Incident and maintenance updates |

## Discord

Primary chat and support channel.

## Guidelines

- Be respectful & constructive
- No sharing of private keys or sensitive data
- Disclose security issues responsibly (see SECURITY.md)
