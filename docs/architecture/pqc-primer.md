# PQC Primer

Post-quantum cryptography protects against future quantum attacks. Dytallix uses lattice-based signatures for accounts and node keys.

> Last updated: 2025-08-20

Key lengths are 48-byte public keys with 32-byte signatures. Tradeoffs include larger payloads and slower verification. The network remains compatible with classical keys for testing.
